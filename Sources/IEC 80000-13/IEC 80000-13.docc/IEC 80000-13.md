# ``IEC_80000_13``

Quantities and units — Part 13: Information science and technology.

## Overview

IEC 80000-13:2008 defines the units of information — the bit (`bit`) and the
byte (`B`, where 1 B = 8 bit) — together with the prefixes for binary multiples,
from kibi (`Ki` = 2¹⁰) through yobi (`Yi` = 2⁸⁰).

```swift
import IEC_80000_13

IEC_80000_13.Unit.byte.bits        // 8   (1 B = 8 bit)
IEC_80000_13.Prefix.kibi.factor    // 1024.0  (2¹⁰)
```
